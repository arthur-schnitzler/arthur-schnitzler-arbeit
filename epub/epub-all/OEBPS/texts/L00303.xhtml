Arthur schnitzler an Richard Beer-Hofmann, 2. 3. 1894

Lieber Richard, sollten sie Anatol brauchen, so kaufen sie gef. auf meine Kosten ein Exemplar; ich müßte das gebundene, das ich habe, als Paket aufgeben, was Umstände macht. Auch kan̅ ich das ungebundene sehr gut brauchen. schade, dass sie nicht schreiben.
Herzl Ihr
Arthur
